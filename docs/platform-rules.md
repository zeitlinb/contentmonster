# Universal Platform Rules (NOT client-specific)

## Output Sizes
| Format | Dimensions | Ratio | Use |
|--------|-----------|-------|-----|
| Feed vertical | 1080x1350 | 4:5 | Instagram/Facebook feed — best performing |
| Feed square | 1080x1080 | 1:1 | Universal fallback, all platforms |
| Stories/Reels | 1080x1920 | 9:16 | Stories, Reels, TikTok |

Every static ad = minimum 4:5 AND 1:1. The 1:1 MUST be repositioned, NOT cropped.

## Upload Resolution
- Render at target dimensions (1080 wide)
- Meta ads: upload at 1440px minimum width to survive compression
- JPG for photo-heavy, PNG for graphic-heavy

## Safe Zones
### Instagram Stories / Reels (9:16)
- Top: 250px no-go (14%)
- Bottom: 340px no-go (20%)
- Safe area: 1080x1610 centered

### Instagram Reels (tighter overlay zones)
- Top: 14%, Bottom: 35%, Sides: 6% each

### Feed (4:5 and 1:1)
- Top/bottom: 180px clear
- All edges: minimum 100px from any text or pill box

## File Size
- Max 5MB for images
- Compress: resample to 2048 width, then JPEG quality 80, then 60 if needed

## Video Specs
- Format: MP4, H.264
- Resolution: match static sizes (1080x1350, 1080x1080, 1080x1920)
- Length: 15 seconds for feed, max 30 seconds
- Design for mute — 83% watch muted
- Captions: mandatory, bold sans-serif, white + black outline, lower third
- ASS subtitles preferred for caption burn-in

## Meta Ads Best Practices
- Headline: 27 chars max
- Primary text hook: first 125 chars
- Headline variants: 3-5 for Advantage+ rotation
- Attribute pills: 2-3 visible on image, mix rational + emotional
- Text coverage: under 20% of image area (Meta penalty threshold)
- No text over faces
