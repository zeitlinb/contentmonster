## Key patterns

1. **Hype dominates engagement; craft dominates substance.** Separate the two before copying anyone — per @X_FINALBOSS (loud) vs. Sean Dollwet (process).
2. **The common stack is converging:** ChatGPT/Claude for prose, Perplexity for research, an image model (Nano Banana/Midjourney) for covers — per @X_FINALBOSS and multiple YT tutorials.
3. **AI-disclosure is now table-stakes** and tools are productizing it (BookWriter's authorship report → KDP form).
4. **Two viable models for us:** authority/audience-first (Dan Koe) vs. KDP-craft/royalty (Sean Dollwet). Revenue-product framing leans toward combining both; avoid the volume-spam lane entirely.
