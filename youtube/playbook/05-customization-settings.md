# Playbook 05 — customization & settings

> Every text field + setting, where it lives, and its limits. Company-agnostic. _Verified 2026-06-28; Studio labels (PUBLISH vs SAVE, tab names) shift between rollouts — confirm live._

## Two locations
- **Customization → Profile** (the public-facing copy): name, handle, description, links, contact email. Click **Publish**.
- **Settings → Channel** (config): keywords, country, feature eligibility. Click **Save**.

> Naming note: what older docs call "Customization → Basic info" is now **Customization → Profile**. "Basic info" is now a **tab under Settings → Channel**.

## @Handle (the URL identity, `youtube.com/@handle`)
- **3–30 chars** (script exceptions exist). Letters/numbers from a supported language + `_ - . ·`; separators **not at start/end**; not case-sensitive; can't look like a URL or phone number.
- Must be unique; one per channel. Change **twice per 14 days**; old handle held 14 days (both URLs work during the hold).
- Set: Customization → Profile → **Handle** → edit → **Publish**.

## Channel name (display name)
- Customization → Profile → **Channel name**. Change **twice per 14 days**.
- ⚠️ **Changing the name removes the verification badge (checkmark).**

## Channel description
- Customization → Profile → **Channel description**. Shown in the About/header.
- ⚠️ Official help states **no specific character limit** for the *channel* description (the 5,000-char limit is for *video* descriptions). ~1,000 is a common community figure; the field shows a live counter — trust that.

## Links
- Up to **14 links** on the channel Home tab (older docs said 5). Each = **title + URL**. The **first** link surfaces near the Subscribe button; the rest under "more."
- Set: Customization → Profile → links section → **Add link**.

## Contact / business email
- Customization → Profile → **Contact info** → email. (Labeled "Contact info," not "business email.")

## Keywords (channel-level — different from per-video tags)
- **Settings → Channel → Basic info → Keywords** → **Save**.

## Country
- **Settings → Channel → Basic info → Country of residence**. Affects YouTube Partner Program eligibility.

## Default video language
- There is **no** "default language" under Settings → Channel. Default upload language = **Settings → Upload defaults → Advanced settings → Video language**. (Separate from your account display language.)

## Verification (phone) — unlock features
- **Settings → Channel → Feature eligibility** (or `youtube.com/verify`) → verify phone.
- Unlocks the **intermediate** tier: **custom thumbnails**, videos **>15 min**, **live streaming** on computer, podcasts.
- **Advanced** features (Content-ID appeals, embeddable live) need stronger (ID/video) verification, not just phone.
- A phone number verifies **≤2 channels/year**. Custom-thumbnail unlock can lag up to ~24h after verifying.
